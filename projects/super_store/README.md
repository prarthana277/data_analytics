# Super Store Analysis

This project contains data files and SQL scripts for analyzing the Super Store dataset.

## Folder Structure

- `data/`  
  Contains raw data files used for analysis.  
  Example: `train.csv` - main dataset.

- `sql/`  
  Contains SQL scripts for data cleaning, exploration, and analysis.  
  Files are numbered to indicate execution order and purpose.

## SQL Scripts Overview

1. **01_data_cleaning.sql**  
   Clean and prepare raw Super Store data.

2. **02_overall_metrics.sql**  
   Calculate overall sales metrics and summaries.

3. **03_sales_by_category.sql**  
   Analyze sales by product category.

4. **05_customer_analysis.sql**  
   Explore customer behavior and lifetime value.

5. **06_product_analysis.sql**  
   Investigate product sales and trends.

6. **07_time_series.sql**  
   Analyze sales trends over time.

7. **08_shipping_analysis.sql**  
   Evaluate shipping times and delays.

8. **09_advanced_window_functions.sql**  
   Use advanced SQL window functions on the dataset.

## How to Use

- Execute the SQL scripts in the order provided to replicate the analysis.
- Modify and expand as needed for your data projects.
